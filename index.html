1-Quem foi Timothy Berners-Lee e qual a sua importância para o HTML?

R: Timothy Berners-Lee, um físico britânico, começou a procurar um meio de, a partir de um documento eletrônico, conseguir navegar para outros documentos que estivessem disponíveis em outros computadores ao redor do mundo. Foi então que Berners-Lee propôs um sistema de hipertexto para o compartilhamento de documentos, que, mais tarde, se tornou a linguagem fundamental da web.

2-Qual necessidade influenciou a criação do HTML?

R: Como o tráfego de dados passou a acontecer entre pessoas de diversas partes do mundo, detentoras de diferentes necessidades e formações, surgiu a demanda por uma padronização da forma como toda essa informação digital poderia ser estruturada. Assim o HTML foi proposto como uma linguagem para a marcação de textos.

3-Explique o papel do W3C para o desenvolvimento do HTML.

R: A organização conhecida como W3C (do inglês World Wide Web Consortium) foi formada, com a intenção de manter um padrão para diferentes navegadores da web. A principal intenção do W3C era que esses navegadores entendessem e processassem as tags HTML de maneira semelhante. (Um dos principais aprimoramentos agregados à linguagem foi o conceito de formulários, que passou a entrar em vigor, embora ainda com tags de marcação muito básicas, como caixas de texto, botões, etc.)

4-Por que o HTML venceu a disputa entre as linguagens de marcação web?

R: Basicamente, o HTML é a unidade essencial de todas as construções da web. Toda página na internet começou meramente como um documento em HTML, com as tags básicas estruturando e formatando esse documento. 

Graças a ideia de Tim Berners-Lee, hoje podemos acessar uma vasta quantidade de informação de qualquer lugar no mundo. Também podemos compartilhar as nossas ideias de forma rápida. Podemos ver noticias, vídeos, fotos e etc.

5-Desenvolva um breve texto sobre a evolução do HTML desde a sua origem até os dias atuais.

R: A história da linguagem de marcação HTML tem seu início em meados da década de 80, quando Timothy Berners-Lee desenvolveu a ideia da World Wide Web (W3C). A primeira versão do HTML, conhecida (HTML 1.0), foi lançada em 1991. Essa versão incluía apenas 22 tags para formatar o texto. Em 1995, o HTML 2.0 foi disponibilizado (embora ainda com tags de marcação muito básicas, como caixas de texto, botões, etc), tava na medida em que a web ganhava cada vez mais popularidade.

À medida que o HTML ganhava a aceitação de um número cada vez maior de pessoas, desenvolvedores passaram a buscar por recursos mais sofisticados para a melhoria de websites. Esse foi o momento em que um grupo de trabalho focado no desenvolvimento do HTML teria apresentado um esboço do que seria a versão 3.0 do HTML. Mas com o tempo, a especificação do HTML 3.0 foi abandonada.

Como o esboço do HTML 3.0 não foi levado adiante e tags proprietárias continuaram surgindo, a padronização da linguagem não podia mais ser adiada. Diante disso, em 1994, o W3C criou e publicou o que ficou conhecido como o HTML 3.2 — o próximo passo na evolução da linguagem. Em janeiro de 1997, o HTML 3.2 se tornou um padrão oficial.

Grandes mudanças vieram com o lançamento do HTML 4.0, versão que representou um grande avanço da linguagem. Em dezembro de 1997, ela foi publicada como a versão recomendada pelo W3C. Em 1999 houve uma versão revisada que foi publicada como HTML 4.01.

A última grande revisão da linguagem desde o lançamento do HTML 4.01 foi o HTML 5. Mais uma vez, essa nova versão trouxe grandes mudanças na maneira como as marcações deveriam ser renderizadas pelos navegadores. Em 2014, o HTML 5.0 tornou-se uma recomendação, e o mesmo aconteceu com o HTML 5.1, em 2016. Finalmente, em 14 de dezembro de 2017, o HTML 5.2 foi lançado como a versão recomendada pelo W3C, a qual prevalece até hoje
